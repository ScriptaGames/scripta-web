# Scripta.co blog

## Dev instructions

 - `npm install -g wintersmith`
 - `git clone https://github.com/ScriptaGames/ScriptaGames.github.io scripta.co`
 - `cd scripta.co`
 - `wintersmith preview`
 - open [localhost:8080](http://localhost:8080)

## Deployment

    make deploy

[Wintersmith](https://github.com/jnordberg/wintersmith#quick-start).
